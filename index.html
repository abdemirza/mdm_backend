API Endpoints

GET /health

Health check endpoint to verify the server is running

Device Management

GET /api/devices

List all devices in the enterprise

GET /api/devices/{deviceName}

Get details of a specific device

POST /api/devices/{deviceName}/lock

Lock a device for a specified duration

Body: {"duration": "120s"}

POST /api/devices/{deviceName}/unlock

Unlock a device (removes any active lock)

POST /api/devices/{deviceName}/lost-mode

Put a device into Lost Mode

Body: {"lostMessage": "This device is lost", "phoneNumber": "+1234567890"}

POST /api/devices/{deviceName}/exit-lost-mode

Disable lost mode on a device

POST /api/devices/{deviceName}/reboot

Reboot a device

POST /api/devices/{deviceName}/reset-password

Reset device password

Enterprise Management

GET /api/enterprise

Get enterprise information

GET /api/enterprise/policies

List all policies in the enterprise

Operations

GET /api/devices/operations/{operationName}

Get the status of a command operation
